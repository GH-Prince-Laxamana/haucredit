# haucredit
Holy Angel University Compliance &amp; Records Engine for Documentation and Institutional Tracking (HAUCREDIT) is a web-based event compliance, checklist, and records management application designed to assist student councils and organizations of Holy Angel University in complying with the Office of Student Affairs (OSA) Student Activity Manual. 

File Structure Descriptions:
1. public - Contains only files that should be accessed by the browser.
2. app - Holds the main application logic.
